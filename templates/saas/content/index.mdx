# Welcome

This is the Markdown source for the home page. Your React marketing page lives in `app/page.tsx` — this file is what AI agents see when they hit `/` with `Accept: text/markdown`.

## Why agent-native?

Most SaaS sites waste 70–90% of an AI crawler's token budget on nav, ads, footers, and JS-rendered content. This template gives agents a clean alternative on every page.

## What's wired up

- Supabase magic-link auth (no passwords)
- Stripe subscriptions (test mode ready)
- @agentready/next middleware for Markdown content negotiation
- Auto-generated `/llms.txt` + `/AGENTS.md`
- `/agent-debug` dashboard
