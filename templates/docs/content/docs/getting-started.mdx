# Getting started

Welcome to your new agent-native docs site. This template ships with:

- A collapsible sidebar that lists every page in `content/docs/`
- Prev/next navigation between pages
- Auto-generated `/llms.txt`, `/AGENTS.md`, and `.md` alternates on every page
- An `/agent-debug` dashboard showing what AI crawlers see

## Install

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Add a page

Create a new file in `content/docs/`:

```
content/docs/my-feature.mdx
```

Give it frontmatter:

```yaml
---
title: My feature
description: What it does.
order: 5
---
```

Save. The sidebar updates automatically.

## Control ordering

Use the `order` frontmatter field. Lower numbers come first in the sidebar.
