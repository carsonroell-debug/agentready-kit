# Welcome

This is an AgentReady starter. The home page React component is in `app/page.tsx`; the Markdown version agents see is this file (`content/index.mdx`).

## What's wired up

- `/llms.txt` — auto-generated from every file in `content/`
- `/AGENTS.md` — instructions for AI crawlers
- `Accept: text/markdown` header → rewrites to the Markdown route
- `/about.md` URL shorthand works too
- `<AgentSection>` and `<AgentMetadata>` components on every page

Edit this file to change the Markdown body that agents see for `/`.
